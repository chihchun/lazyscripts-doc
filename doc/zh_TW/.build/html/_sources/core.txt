主程式(Core)
============

你可以把主程式看做是一個Thin Client。

Thin clinet 負責的任務為：

#. 讀取腳本清單(Script List)，設定腳本庫位置，取得腳本來源。
#. 更新腳本庫裡的腳本版本。
#. 顯示提供的腳本，執行使用者選擇的腳本進行安裝，或客製化。

=====================
如何開發主程式(Core)
=====================

為什麼我們用 Python 呢? 因為愛!
為什麼我們用 Git 呢? 因為愛!

**準備開發環境** ::

    user@host$ make init_devenv

**執行自我測試**::

    # 不顯示錯誤。
    user@host$ make test

    # 顯示錯誤
    user@host$ make devtest
